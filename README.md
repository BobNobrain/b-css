# TBEM

> A simplier way for writing BEM-oriented CSS

[Documentation](docs/index.md) (will appear there in some time)

## Examples

```
block todoList
    list-style none
    padding 10px
    margin 0

    el item
        display block
        padding 8px

        mod done: boolean
            opacity 0.5

    el addButton
        margin 0 auto
```
